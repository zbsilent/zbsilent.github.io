TIP

关于虚拟 DOM：
1.本质是 Object 类型的对象 （一般对象）
2.虚拟 DOM 比较“轻”， 真实 DOM 比较“重”， 因为虚拟 DOM 是 React 内部在用， 无需真实 DOM 上那么多的属性。
3 虚拟 DOM 最终会被 React 转化为真实 DOM，呈现在页面上。

TIP

jsx 语法规则：
1.定义虚拟 DOM 时，不要写引号。
2.标签中混入 JS 表达式时要用{}。
3.样式的类名指定不要用 class， 要用 className。ES6 关键字
5.只有一个根标签
6.标签必须闭合
7.标签首字母
（1）.若小写字母开头，则将改标签转为 htm1 中同名元素，若 htm1 中无该标签对应的同名元素，则报错。
（2）.若大写字母开头，react 就 去渲染对应的组件，若组件没有定义，则报错。

WARNING

定注意区分：[js语句（代码）]与[js表达式]
1.表达式：一个表达式会产生一个值，可以放在任何一个需要值的地方
下面这些都是表达式：
(1). a
(2). a+b
(3). demo(1)
(4).arr.map()
(5). function test () {}
2.语句（代码）：
下面这些都是语句（代码）：
(1).if(){}
(2).for(){}
(3).switch(){case:xxxx}

WARNING

(1)map()会分配内存空间存储新数组并返回，forEach()不会返回数据。
(2)forEach()允许callback更改原始数组的元素。map()返回新的数组
